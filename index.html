The Hardy-Weinberg Equilibrium Model

In population genetics, the Hardy-Weinberg Equilibrium Model states that allele and genotype frequencies in a population will remain constant from generation to generation in the absence of other evolutionary influences.

Glossary

Evolution
The change in allele frequencies in a population over time.
Allele
Version of a gene, e.g. the gene for eye colour.
Genotype
The combination of two related alleles in a cell.
Phenotype
The expression of the gene in the organism, e.g. brown eyes.
Allele frequency
The proportion of alleles in a gene pool that are either dominant, e.g. code for brown eyes, or recessive, e.g. code for blue eyes.
Genotype frequency
The proportion of allele combinations in a cell.
Homozygous dominant chromosome
Both the allele inherited from the mother and the one inherited from the father are dominant.
Homozygous recessive chromosome
Both the allele inherited from the mother and the one inherited from the father are recessive.
Heterozygous chromosome
One allele is dominant and one is recessive.

Forces that lead to changes in allele frequencies

  1. Genetic drift
  2. Mutation
  3. Migration
  4. Selection (natural or otherwise)

The Hardy-Weinberg model is a null model

A null model establishes a baseline against which changes in a system can be measured after the application of a force.

Assumptions

  • None of the the four forces listed above are brought to bear on the model: in effect, evolution does not take place.
  • There is random sexual reproduction (no sexual selection).
  • There is an infinite population size (therefore it is not susceptible to genetic drift).
  • Generations are non-overlapping, i.e. when a new generation is born, the old generation dies out.
  • Body cells are diploid.

Possible genotypes in a diploid cell

By convention:

  • P represents the dominant allele, and
  • q represents the recessive allele.

To keep things trackable, we focus on one gene, e.g. eye colour, where the possible alleles are:

  • P = brown eyes (dominant)
  • q = blue eyes (recessive)

  • PP = homozygous dominant, brown eyes.
  • Pq = heterozygous, brown eyes (the dominant allele will always trump the recessive allele).
  • qq = homozygous recessive, blue eyes.

Calculating genotype frequencies

We can calculate genotype frequencies for a population only if we know the percentage of that population in which the homozygous recessive genotype (qq) has been expressed. (The PP percentage could not be derived from empirical observation, because the Pq genotype is also P dominant.)

The Hardy-Weinberg equation

The Hardy-Weinberg equation is used to measure genotype frequencies in a population.

P2 + 2Pq + q2 = 1

If we have the value of qq i.e. q2, we can solve the equation and get the genotype frequencies for PP and Pq as well:

  • √q2 = q
  • 1 − q = P
  • P2 = P × P
  • 2Pq = 2(P × q)

The genotype frequencies will not always sum to 1 (100%) due to rounding — this is expected behaviour.

Checking the result

This can be done by calculating the allele frequencies and adding them together.

  • P2 + (2Pq ÷ 2) = P frequency.
  • (2Pq ÷ 2) + q2 = q frequency.

P frequency + q frequency must sum to 1 (100%).

Phenotype frequencies

The values of the phenotype frequencies, i.e. the expression of the genotypes (in this case blue or brown eyes) in a population, are as follows:

  • Blue eyes = qq (This is a given: it is the number we had to work with from the beginning).
  • Brown eyes = PP + 2Pq
